# synusiaLookup
Ultra bàsic Lookup de isbn a partir de títol, autor i publisher

## Usage
Entrada de fitxer csv:
```
$ cat titleAuthor.csv
author,title,publisher
Jack London, Martin Eden,akal
Alexander Berkman, El mito bolchevique,
```
Sortida a **titleAuthor-out.csv**
